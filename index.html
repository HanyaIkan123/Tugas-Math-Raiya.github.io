! TUgas Math JS

Math.Round => Membulatkan Nilai ke angka terdekat

Jika angka desimal diatas atau sama dengan 0.5 maka akan dibulatkan keatas
namun jika angka desimal dibawah 0.5 maka akan dibulatkan kebawah

Variabel

A = 7.49

B = 10.51

c = 3.75

Hasil

Hasil pembulatan 7

Hasil pembulatan 11

Hasil pembulatan 4

Math.floor => membulatkan angka desimal kebawah

apapun angka desimalnya seperti 2.99 maka akan dibulatkan menjadi angka 2

Variabel

A = 5.99

B = 6.76

c = 4.5

Hasil

Hasil pembulatan 5

Hasil pembulatan 6

Hasil pembulatan 4

Math.ceil => membulatkan angka desimal keatas

Berapapun angka desimalnya seperti 5.99 akan menjadi 6

Variabel

A = 2.01

B = 9.5

c = 6.4

Hasil

Hasil pembulatan 3

Hasil pembulatan 10

Hasil pembulatan 7

Math.pow => mencari hasil pemangkatan

ya pangkat apalagi emang ?

Variabel

A = 2

B = 3

c = 4

Hasil

Hasil pemangkatan (2 pangkat 3) = 8

Hasil pemangkatan (3 pangkat 4) = 9

Hasil pemangkatan (2 pangkat 4) = 16
